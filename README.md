# 🎵 Northcoders Record Shop API

Welcome to the **Northcoders Record Shop** — where the vinyl spins, the shelves are stacked, and the inventory system has finally escaped the dark ages of pen and paper.

This project is a backend REST API built to manage record shop inventory, track album stock, and keep everything organised without relying on scribbled notes and crossed fingers.

Built with **ASP.NET Core**, **Entity Framework**, and a healthy amount of caffeine.

---

# 🎯 Project Goals

The aim of this project was to build a production-quality backend API that can:

✅ Store album information  
✅ Track inventory and stock levels  
✅ Perform full CRUD operations  
✅ Support album searching and filtering  
✅ Handle errors gracefully  
✅ Follow clean architecture principles  
✅ Be fully unit tested using TDD  

---

# 🎸 Features

## Core Features (MVP)

- 🎶 View all albums in stock
- 🔍 Get album details by ID
- ➕ Add new albums
- ✏️ Update album information
- ❌ Delete albums
- ❤️ `/health` endpoint for API health checks

---

## Bonus Features

- 🎤 Filter albums by artist
- 📀 Filter albums by genre
- 📅 Filter albums by release year
- 🔎 Search albums by album name
